1 Why Emacs?

1.1 Flexibility

Emacs Lisp makes it a joy to extend, attracting a large community of lisp hackers. Case in point: Amazon's Customer Service application Mailman was built in Emacs Lisp.

https://sites.google.com/site/steveyegge2/tour-de-babel

1.2 Uniform Interface

Everything is text, everything is in a buffer. Keyboard movements work everywhere, no surprises.

1.3 Emacs has everything you need

  • Web Browser (eww)
  • IRC Client (erc/irc)
  • Email Client (mu4e/notmuch/gnus)
  • Emacs X Window Manager (exwm)
  • File Browser (dired)

3 Undo

There is no redo. Add a non-editing command to undo the undo, eg. `C-g`. In that way `C-g C-_` performs a redo.

http://stackoverflow.com/questions/3527142/how-do-you-redo-changes-after-undo-with-emacs
